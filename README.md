# Descripción

Sistema de e-commerce de ropa con **menú interactivo por consola**, implementado completamente con **programación funcional** en Java, utilizando:

- ✅ **Menú interactivo recursivo** (sin ciclos while/for)
- ✅ **Interfaces funcionales** (personalizadas y predefinidas)
- ✅ **Expresiones lambda**
- ✅ **Streams API**
- ✅ **Recursión** (sin uso de ciclos for/while)
- ✅ **Inmutabilidad** de datos
- ✅ **Composición funcional**
- ✅ **Optional** para manejo de valores nulos
